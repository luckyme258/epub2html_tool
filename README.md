# epub2html_tool
EPUB to HTML Converter - A GUI tool to convert EPUB eBooks to HTML files. / EPUB转HTML图形化工具 - 将EPUB电子书转换为HTML文件

## 📚 EPUB2HTML Tool v1.0.0 (Windows) / EPUB转HTML工具 v1.0.0 (Windows版)

### ✨ Features / 功能特点

| English | 中文 |
|---------|------|
| 🖥️ **Graphical Interface** - Easy to use | 🖥️ **图形化界面** - 操作简单直观 |
| 📖 **Smart Parsing** - Preserve book structure | 📖 **智能解析** - 保持原书结构 |
| 🔧 **Tool Extraction** - Auto-extract tools from chapters | 🔧 **工具提取** - 自动提取章节中的工具 |
| 📝 **Chapter Summary** - Auto-extract summaries | 📝 **章节小结** - 自动提取小结内容 |
| 🌓 **Theme Switch** - Day/Night mode | 🌓 **主题切换** - 支持日间/夜间模式 |
| 📱 **Mobile Friendly** - Responsive HTML | 📱 **移动端适配** - 支持手机浏览 |
| 📊 **Reading Progress** - Real-time progress | 📊 **阅读进度** - 实时显示进度 |

### 🚀 How to Use / 使用方法

**English**  
1. Download `EPUB转HTML工具.exe`  
2. Double-click to run  
3. Click "Browse" to select an EPUB file  
4. Click "Start Conversion"  
5. The generated HTML file will open automatically  

**中文**  
1. 下载 `EPUB转HTML工具.exe`  
2. 双击运行  
3. 点击“浏览”选择EPUB文件  
4. 点击“开始转换”  
5. 自动打开生成的HTML文件
